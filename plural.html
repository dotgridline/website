What's this?

Plural was my final project at university. It was done in one year and a half and had the subject of usability, ubiquity and the evolution of the Multi-touch technology.

What's the idea behind it?

The idea of Plural was to be a system developed for collaboration between multiple users, in a multi-touch tabletop, using technologies like NFC and Cloud computing to improve workflows and allow a ubiquitous use of technology.

The Design process

It was developed by me from the ground, starting with the information architecture of the system, and them going to tests with users for testing the usability of the system. After several tests, problems and solutions, i came up with a system, visually inspired in a flat, minimal design, that allowed fast interactions, with lot of touch gestures and simultaneous interaction between users.

How it would work

When the user enters the "meeting room", his cell phone will connect via NFC to the Tabletop, and using the Wi-Fi of the room, it will sync every project the user wanted to use as a base for the collaboration. Then, all the users could use the built-in tools to create –– and modificate everything they could do with pens , papers and much more. Then, the "mutual" work could be easily shared via NFC and Wi-Fi to the other phones in the room.

What makes this project so special?

  • It was all developed from the ground with one clear objective: Study to improve the Touch "language".
  • It is focused in creation of content – Touch technology is much more focused in consuming and not in creating content.
  • The technologies used and the concepts made using future technologies to make it portable, just like a sheet of paper.
  • Based on the concepts of Natural User Interface(NUI) to make it easy and intuitive.
  • It's ubiquitous, It's the future.
