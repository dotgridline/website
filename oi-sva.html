More Info

Oi SVA is a project where we created a site for the download of music, wallpapers and other types of media for the telecommunication company called Oi. It was the most complete, cross-browser and cross-device mobile project i have done.

The complete project was made of:

  • 2 WML versions, with and without images;
  • 4 Low XHTML – 128px, 176px, 240px and 320px – made only with tables, almost no css;
  • 4 Mid XHTML Mobile Profile – same sizes as the low version – made using css to style elements and no javascript.
  • 1 High HTML5 Touch - It was responsive, with plenty of css3, javascript and touch gestures.

It was our first website created to work as a WebApp and it has been a great success for both companies.
